# Platform Rules (Quy tắc riêng từng nền tảng)

> **AI: Luôn đọc file này trước khi format/rewrite bài viết, để đảm bảo output phù hợp đặc thù từng nền tảng.**

---

## 🔵 LinkedIn

### Formatting Rules

- Dòng đầu tiên (Hook) phải **mạnh, gây tò mò** → Người dùng thấy 2 dòng đầu trước khi bấm "See more".
- Xuống dòng **sau mỗi 1–2 câu** (LinkedIn ăn khoảng trắng, đọc dễ hơn).
- Ít dùng bullet point (`-`) quá nhiều vì LinkedIn không render đẹp.
- **KHÔNG** thêm link trực tiếp trong bài (LinkedIn giảm Reach bài có link ngoài). Để link ở comment.
- Kết thúc bằng 1 câu hỏi mở hoặc CTA cụ thể.

### Content Rules

- Ưu tiên: Bài chia sẻ kinh nghiệm cá nhân (First-person), Lesson Learned, số liệu thực.
- Hashtag: 3–5 cái, đặt cuối bài. Ví dụ: `#IndieHacker #iOSDev #BuildInPublic`

## 🐦 X / Twitter (Thread Format)

### Formatting Rules

- Tweet 1 = Hook cực ngắn, gây tò mò ngay, hoặc đưa ra 1 claim bất ngờ.
- Mỗi tweet ≤ 280 ký tự.
- Đánh số thread: `1/`, `2/`, `3/`... ở cuối mỗi tweet.
- Tweet cuối cùng: CTA + repost/follow request.
- Không dùng ký tự Unicode lạ (dấu đặc biệt, emoji box drawing) dễ bị lỗi hiển thị.

### Content Rules

- Hot takes được phép. Controversial OK nếu có dữ liệu backup.
- Thread ngắn: 5–7 tweets. Thread dài: tối đa 12 tweets.

## 📰 Medium (Markdown → Rich Text)

### Formatting Rules

- Dùng `# H1` cho Title (chỉ 1 cái duy nhất).
- Dùng `## H2` cho các section chính.
- Hình ảnh: đặt sau paragraph quan trọng để minh họa.
- `---` để tách section lớn.

### Content Rules

- Bài Medium cần có: Intro (problem setup) → Body (phân tích + ví dụ) → Conclusion (Takeaways).
- Độ dài lý tưởng: 800–2000 từ.
- Canonical URL: Nếu bài đã đăng trên blog khác, set canonical để tránh SEO penalty.

---

## 📘 Facebook (Tùy chọn)

### Formatting Rules

- Facebook **không render Markdown**. Script cần strip toàn bộ syntax Markdown (`**bold**`, `#heading`) trước khi copy.
- Ảnh đầu tiên là QUAN TRỌNG NHẤT (Facebook ưu tiên post có ảnh đẹp).
- Dòng đầu tiên vẫn phải là Hook.

### Content Rules

- Giọng văn nhẹ hơn, thân thiện hơn LinkedIn.
- Hạn chế tiếng Anh kỹ thuật, giải thích bằng tiếng Việt đơn giản.
